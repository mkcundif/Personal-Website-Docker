In this case study, I analyzed the strategic and financial implications of a major bank’s decision to invest heavily in artificial intelligence. Corporate Programs Management System

Developed a fully functional Corporate Programs Management System that integrated a Python backend, SQL database, and HTML/CSS front end to manage courses, sections, and student enrollments. The system allowed users to add and manage courses, enroll students, and generate dynamic reports displaying course sections and enrollment statistics. This project demonstrated full-stack development skills by connecting database logic, backend functionality, and an interactive web interface for efficient academic program management.
